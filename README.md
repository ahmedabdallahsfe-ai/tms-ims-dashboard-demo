# Sell-In vs Sell-Out Control Tower (Demo)

Interactive pharma BI dashboard comparing TMS (sell-in to distributor) vs IMS (sell-out to market).

**Features:** multi-select filters (Month, BU, Line, Brand, SKU, Sales Type) - QTY/Value toggle - KPI cards - brand comparison and monthly trend charts - risks and opportunities engine - Channel Inventory tab with implied inventory, months of coverage, and stockout projections.

Single-file app (`index.html`) - React + Recharts + Tailwind via CDN. No build step.

*Demo data: brand and product names are anonymized and all figures are scaled - patterns are real, absolute numbers are not.*
